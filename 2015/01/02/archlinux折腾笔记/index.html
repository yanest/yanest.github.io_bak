一直听说archlinux挺折腾，一直没敢尝试。但是心里面总是很痒痒，正好赶上元旦放假，媳妇串休，在家呆着没事，折腾一下arch linux。

安装

安装介质

直接去官网下了一个最新的iso，也可以使用usb安装

进入安装环境

使用下载回来的ISO引导系统，配置好网络。超简单，不多说。

分区

cfdisk #分区工具，也可以使用fdisk
lsblk -f #查看分区信息
mkfs.ext4 /dev/sda1 #格式化分区
mount /dev/sda1 /mnt #挂载系统

配置安装源

rm /etc/pacman.d/mirrorlist
nano /etc/pacman.d/mirrorlist #添加网易源 Server = http://mirrors.163.com/archlinux/$repo/os/$arch

开始安装

pacstrap -i /mnt base base-devel
pacman-key --init && pacman-key --populate archlinux #若 pacman 报告 error: failed to commit transaction (invalid or corrupted package), 执行此命令

配置系统

genfstab -U -p /mnt >> /mnt/etc/fstab #生成fstab
arch-chroot /mnt /bin/bash #chroot到系统
# nano /etc/locale.gen
en_US.UTF-8 UTF-8
zh_CN.UTF-8 UTF-8
zh_TW.UTF-8 UTF-8
locale-gen #生成指定的本地化文件
echo LANG=en_US.UTF-8 > /etc/locale.conf #将系统 locale 设置为en_US.UTF-8
ln -s /usr/share/zoneinfo/Asia/Shanghai /etc/localtime #配置时区
hwclock --systohc --utc
pacman -S grub #安装grub
grub-install --target=i386-pc --recheck /dev/sda
systemctl enable dhcpcd@interface_name.service
grub-mkconfig -o /boot/grub/grub.cfg
